Detailed Description

Diese Klasse wird verwendet, um eines neues mit der Klasse Card zu erstellen, damit danach ein neues Stack für das Spiel erstellen kann.

Constructor & Destructor Documentation

Blackjack.Logic.Deck.Deck ( )

Dies ist der Konstruktor. Hier werden die ganzen Karten erstellt für das Deck. Das heißt, dass 2 for-Schleifen verwendet werden, eine ist für die Erstellung der Art der Karten und die andere für die Werte. Die erste ist für die Art und muss deswegen 4 durchlaufen, wegen (Karo, Pik, Kreuz, Herz). Die 2. muss 13 durchlaufen werden.
