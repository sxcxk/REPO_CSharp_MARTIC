Detailed Description

Diese Klasse wird dazu verwendet, um eine Karte zu erstellen. Auch werden hier der richtige Wert bzw. das richtige Bild für die Karte zugewiesen. Dazu ist diese Klasse die Super-Klasse für die Klasse BlackjackCard.
